# Semeel jak - Means "forest" in Kiliwa language.

Web app that bridges open research data with the gbif api and recommends datasets, taxa and geographies for novel research.

## Libraries
* Django
* Pillow
* Bootstrap
* JQuery
* Tesseract (pytesseract)
* Celery
* Tensorflow
